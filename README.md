# Gerenciador de Tarefas

## Desenvolvido por
- Kawe Lobo
- Kaua Igergryam

## Descrição
Sistema CRUD simples desenvolvido em Java utilizando:
- Estruturas condicionais
- Laços de repetição
- Programação Orientada a Objetos

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Atualizar tarefas
- Remover tarefas

## Como executar
1. Abra em uma IDE Java
2. Execute Main.java
3. Utilize o menu no terminal
